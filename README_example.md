- Postgres
- Git
- Ruby 2.3.0
- Bundle

# Installation

Quick setup to get you started

```bash
git clone git://github.com/my_repo.git
cd my_repo
bundle
bundle exec rake db:create db:migrate
```

# Running Tests

Here's how to run tests

```bash
bundle exec rspec
```
